# Agent Instructions

This repository uses the `vibe-coding-standard-workflow` process.

Before writing code, read:

- `memory-bank/architecture.md`
- `memory-bank/design-document.md`
- `memory-bank/implementation-plan.md`
- `memory-bank/progress.md`

Execute exactly one implementation-plan step at a time unless the user explicitly changes the workflow. After each validated step, update:

- `memory-bank/progress.md`
- `memory-bank/architecture.md`

Keep `memory-bank/` as the source of truth for product intent, implementation order, and architecture.

## 重要提示

- 写任何代码前必须完整阅读 memory-bank/@architecture.md
- 写任何代码前必须完整阅读 memory-bank/@design-document.md
- 每完成一个重大功能或里程碑后，必须更新 memory-bank/@architecture.md
